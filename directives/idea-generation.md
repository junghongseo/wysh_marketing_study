# SOP: 마케팅 아이디어 생성 (Idea Generation)

> **버전**: v1.0  
> **역할**: 챕터 분석 결과를 바탕으로 WYSH 맞춤 마케팅 아이디어를 생성하고 MFS로 채점하는 절차  
> **선행 조건**: Step 2~5 (트랜스크립트, 챕터 분석, WYSH 컨텍스트, 트렌드 리서치)가 완료되어야 함

---

## 목적

"This is Marketing" 챕터의 핵심 원칙과 WYSH의 현재 브랜드 상황을 결합하여, 즉시 실행 가능한 "Smallest Viable Action" 3~5개를 도출한다.

---

## 입력 데이터

에이전트는 다음 파일을 **반드시** 읽은 후 아이디어를 생성해야 한다:

| 파일 | 읽는 이유 |
|------|-----------|
| `data/weeks/week-XX/chapter-analysis.md` | 이번 주 챕터의 핵심 원칙 파악 |
| `data/weeks/week-XX/transcript.md` | 원본 텍스트에서 추가 인사이트 확인 |
| `data/weeks/week-XX/wysh-context.json` | WYSH의 현재 제품/프로모션/콘텐츠 상황 |
| `data/weeks/week-XX/trends.md` | 시장 트렌드와의 정합성 확인 |
| `data/weeks/week-(XX-1)/ideas.json` | 이전 주차 아이디어 (중복 방지) |

## 아이디어 생성 프레임워크

### 1. "Smallest Viable Action" 원칙

Seth Godin의 "Smallest Viable Market" 개념을 액션 레벨로 축소한다:
- **작은 시작**: 1주일 내에 실행할 수 있는 크기
- **측정 가능**: 결과를 정량적으로 평가할 수 있음
- **WYSH 맞춤**: 무지방 고단백 그릭요거트 D2C 브랜드에 구체적으로 적용

### 2. 아이디어 카테고리

아이디어는 반드시 다음 카테고리 중 하나에 속해야 한다:

| 카테고리 | 설명 | 예시 |
|----------|------|------|
| 📱 **콘텐츠** | 인스타그램/블로그 콘텐츠 | 제품 활용 숏폼, 고객 후기 카드 |
| 🎯 **타겟팅** | 특정 고객군 공략 | 다이어트 커뮤니티 진입, 헬스 인플루언서 협업 |
| 💡 **브랜딩** | 브랜드 인식 강화 | 슬로건 실험, 패키지 메시지 변경 |
| 🛒 **전환** | 구매 전환율 향상 | 랜딩페이지 CTA 변경, 첫 구매 혜택 |
| 🔄 **리텐션** | 재구매 유도 | 정기배송 혜택, 고객 커뮤니티 |

### 3. MFS (Marketing Feasibility Score) 채점

각 아이디어를 `marketing-ideas` 스킬의 MFS 기준으로 채점한다:

```
MFS = (Impact + Fit + Speed) − (Effort + Cost)

각 항목: 1~5점
Score Range: -7 ~ +13
```

| 점수 | 의미 | 액션 |
|------|------|------|
| 10~13 | 극히 높은 레버리지 | 즉시 실행 |
| 7~9 | 강한 기회 | 우선순위 부여 |
| 4~6 | 상황적으로 유효 | 선택적 테스트 |
| 1~3 | 한계적 | 보류 |
| ≤0 | 부적합 | 추천하지 않음 |

**WYSH 맞춤 가중치**:
- **Fit**: WYSH의 D2C 그릭요거트 포지셔닝과의 적합도를 특히 엄격하게 평가
- **Speed**: 소규모 팀 기준 실행 속도 평가 (1명이 1주 내에 가능한가?)
- **Cost**: 무/저예산 우선 (월 50만원 이하)

## 출력 형식

아이디어는 반드시 아래 JSON 형식으로 `data/weeks/week-XX/ideas.json`에 저장한다:

```json
{
  "week": 1,
  "chapter": "Chapter 1: ...",
  "generated_at": "2026-02-14T10:00:00+09:00",
  "ideas": [
    {
      "id": 1,
      "title": "아이디어 제목",
      "category": "콘텐츠",
      "description": "구체적인 실행 설명",
      "chapter_principle": "이 아이디어가 기반하는 챕터 원칙",
      "smallest_viable_action": "1주 내 실행 가능한 최소 액션",
      "success_metric": "측정 가능한 성공 지표",
      "mfs": {
        "impact": 4,
        "fit": 5,
        "speed": 4,
        "effort": 2,
        "cost": 1,
        "total": 10,
        "recommendation": "즉시 실행"
      },
      "risks": ["잠재 리스크 1"],
      "reference_trend": "관련 트렌드 (trends.md 참조)"
    }
  ],
  "summary": "이번 주 아이디어 요약 (2~3문장)"
}
```

## 마크다운 출력

JSON 외에 사람이 읽기 편한 마크다운도 함께 생성한다:

```markdown
# Week X 마케팅 아이디어

## 📊 요약
이번 주는 [챕터명]의 [핵심원칙]을 바탕으로 X개의 아이디어를 도출했습니다.

## 🏆 추천 아이디어

### 1. [아이디어 제목] (MFS: +10)
**카테고리**: 콘텐츠 | **추천**: 즉시 실행

**핵심**: [설명]

**실행 방법**: [구체적 단계]

**성공 지표**: [측정 가능한 KPI]
```

## 품질 가이드라인

- ❌ **추상적 제안 금지**: "브랜딩을 강화하세요" → ❌
- ✅ **구체적 액션**: "인스타그램에 고객 다이어트 성공 후기를 카드뉴스 형식으로 주 2회 게시" → ✅
- ❌ **아이디어 5개 초과 금지**: 결정 피로를 방지
- ✅ **MFS 4점 이상만 추천**: 실행 가능성이 낮은 아이디어는 제외
- ✅ **이전 주차와 중복 금지**: 지난 주 ideas.json을 반드시 확인
